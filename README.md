# collabstage

# Setup
Install dependencies
```
cd client
npm install
```

```
cd server
npm install
```

## Running the dev client
```
cd client
npm start
```
## Running the server
```
cd server
npm start
```

## Debug
Mobile debugging setup
#### Forwarding ports
```
sudo ufw allow 3000
sudo ufw allow 9000
```

#### Chrome dev tools
chrome://inspect/#devices
